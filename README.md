# Pathophys Exam Simulator v19 Editable + Add Questions

Upload all files in this folder to GitHub Pages.

New features:
- Edit a question during the quiz; the timer pauses while editing.
- Add a new question during the quiz or from setup.
- Add/remove answer choices and mark correct choices.
- Add an image by URL/path, or upload an image from your device.
- Export/import your added questions and edits as JSON backups.

Browser-saved edits and added questions are stored locally in your browser, so export backups regularly.
